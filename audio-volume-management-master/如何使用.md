# 如何使用

## 方法一：在 DevEco Studio 中直接运行（推荐）

**步骤：**
1. 打开 DevEco Studio
2. 连接设备（真机或虚拟机）
3. 点击运行按钮（绿色三角形）

**说明：**
- DevEco Studio 会在运行前自动执行卸载脚本
- 自动解决签名不一致问题
- 无需任何手动操作

**如果仍然出现签名问题：**
- 在设备上手动卸载应用一次
- 然后在 DevEco Studio 中点击运行

## 方法二：使用独立部署脚本

**步骤：**
1. 双击 `独立部署.bat`
2. 等待部署完成

**说明：**
- 完全独立，不依赖 DevEco Studio
- 自动卸载、构建、安装、启动

## 推荐使用

**日常开发：** 在 DevEco Studio 中点击运行

**快速测试：** 双击 `独立部署.bat`
